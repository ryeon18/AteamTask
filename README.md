### ▶︎ 설치 및 실행 방법
1. 터미널에서 해당 프로젝트를 clone하고, 프로젝트 폴더로 이동한다.
```shell
git clone https://github.com/ryeon18/AteamTask.git
code .
```

2. 프로젝트에 사용한 package를 설치한다
```shell
npm install
```

3. 다음 명령어로 서버를 실행시킨다
```shell
npm start
```
